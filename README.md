# Hospitality Performance Dashboard (Power BI)

## Overview
This project presents a Power BI dashboard built to monitor key hospitality performance metrics such as Revenue, RevPAR, ADR, and Occupancy%.

The goal of the dashboard is to provide a clear, high-level view of hotel performance across properties, cities, and time periods.

## Metrics Covered
- Revenue
- RevPAR
- ADR
- Occupancy %
- Daily Sellable Night Rooms (DSNR)

## Dashboard Features
- City-level and property-level filtering
- Weekday vs weekend performance comparison
- Trend analysis across time
- Interactive slicers for exploratory reporting

## Tools Used
- Power BI
- Excel (data preparation)

## Project Type
Business Intelligence | Reporting Dashboard
